# Odin Recipes

This project is part of The Odin Project's Foundations course (Project: Recipes). The goal is to practice basic HTML by building a simple recipe website before moving on to CSS.

## Live Preview

- [Live Preview](https://ben-kw.github.io/odin-recipes/)

## Project Overview

This website is a small, multi-page recipe site with a main index page linking to individual recipe pages. It focuses on clean, semantic HTML and readable structure.

## What the Website Includes

- A main index page
- Multiple recipe pages
- Images and descriptions
- Navigation between pages

## Skills I Practiced

- Structuring a multi-page website
- Using headings, paragraphs, lists, and images
- Organizing files and linking between pages
- Writing clean, semantic HTML
- Building readable markup with no CSS
